# Banking-Dashboard

1. Title
Banking Client Portfolio Dashboard

2. Short Description
This multi-page Power BI dashboard provides a 360-degree analysis of the bank's client portfolio, detailing key loan and deposit metrics. It offers an executive-level summary and deep-dive segmentation by client demographics, income, and engagement.

3. Features & Highlights
-Multi-Page Navigation: The report is organized into three distinct, user-friendly sections: a high-level Summary, a Loan Analysis deep-dive, and a Deposit Analysis  deep-dive.
-Executive KPI Summary: The "Summary" tab displays 12 key performance indicators (KPIs) in one view, including Total Clients (1,333), Total Loan ($1.99bn), Total  Deposit ($1.73bn), and Total Fees ($72.6M).
-Loan Portfolio Deep-Dive: The "Loan Analysis" tab (Image 2) breaks down loan products by:
-Type: Bank Loan ($814.22M), Business Lending ($1.17bn), etc.
-Client Segment: Income Band, Nationality, and Engagement.
-Deposit Portfolio Deep-Dive: The "Deposit Analysis" tab (Image 3) provides a similar breakdown for deposits, segmented by:
-Account Type: Bank Deposit ($925.4M), Savings ($324.13M), Checking ($434.77M).
-Client Segment: Income Band, Nationality, and Engagement Timeframe.
-Interactive Filtering: Features powerful top-level filters for Banking Relationship (e.g., Private Bank), Gender, Investment Advisor, and various Time Periods (Last 30D, Last 12M, etc.) to analyze specific cohorts.
